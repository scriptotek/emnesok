About

The Subject Search is developed by the University of Oslo Science Library. Vocabulary search and retrieval is powered by a Skosmos instance using data from data.ub.uio.no, while library catalogue search and retrieval is powered by Primo through a locally developed middleware.

The Subject Search is open source and licensed under the MIT license.

For information about the subject vocabularies, see the start page.

Privacy

So far we're not using cookies or localStorage at all. If we introduce login or user preferences in the future, we will need storage, but we will always limit usage to a minimum.

We use https to ensure anonymous searching.

For analytics, our initial plan was to use the open source tool Piwik, but we were persuaded into using Google Analytics which is pervasive at the university. Note that two extra steps have been taken to protect your privacy:

  • We're proxying all data through ipproxy, which masks the last part of your IP address, meaning that your full IP address is never sent to any third party (Google).
  • As an experiment we've disabled all GA cookies. While we need some statistics on visits and usage, we believe that tracking users is not really needed for this tool. Users tracking also has a lot of flaws.
